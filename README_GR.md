# Οντοκεντρικός Προγραμματισμός – Εργαστηριακό Υλικό & Project (2019 -2020)

Το αποθετήριο περιέχει το υλικό του μαθήματος **«Οντοκεντρικός Προγραμματισμός»**  
του 2ου εξαμήνου στο **Τμήμα Μηχανικών Η/Υ & Πληροφορικής (CEID), Πανεπιστήμιο Πατρών**.

Το υλικό αποτελείται από τρία σετ εργαστηριακών ασκήσεων και ένα μικρό project  
(σε Java ή C++), τα οποία υλοποιήθηκαν κατά τα πρώτα στάδια των σπουδών μου.

---

## 📚 Περιεχόμενα

### **1. Java – Lab Set 1**
**Θεματολογία:**
- Βασική σύνταξη Java  
- Command-line arguments  
- Συνθήκες και δομές ελέγχου ροής  
- Εισαγωγή σε κλάσεις και αντικείμενα  

**Φάκελος:** `labs/java-set-1/`

---

### **2. Java – Lab Set 2**
**Θεματολογία:**
- Κλάσεις, κληρονομικότητα και πολυμορφισμός  
- Abstract classes και overriding  
- Συλλογές (ArrayList, iterators, generics)  
- Χειρισμός εξαιρέσεων (custom exceptions)  
- File I/O (ανάγνωση και εγγραφή αρχείων κειμένου/δυαδικών)  

**Φάκελος:** `labs/java-set-2/`

---

### **3. C++ – Lab Set 3**
**Θεματολογία:**
- Βασική σύνταξη C++ και I/O  
- Πίνακες & πολυδιάστατοι πίνακες  
- Κλάσεις, constructors και access specifiers (private, protected, public)  
- Operator overloading  
- Κληρονομικότητα & virtual functions  
- STL `vector` και iterators  

**Φάκελος:** `labs/cpp-set-3/`

---

### **4. Τελικό Project (θα προστεθεί)**
Το μάθημα περιλάμβανε και ένα τελικό project, το οποίο μπορούσε να υλοποιηθεί  
σε **Java** ή **C++**.  
Η δική μου υλοποίηση έγινε σε **Java** και βρίσκεται στον φάκελο:

`project/java/`

---

## 🎯 Σκοπός του Αποθετηρίου

Το παρόν αποθετήριο στοχεύει να:

- καταγράψει την πρώτη μου εμπειρία στον αντικειμενοστραφή προγραμματισμό σε **Java** και **C++**  
- παρουσιάσει βασικές έννοιες όπως:  
  - σχεδίαση κλάσεων & ενθυλάκωση  
  - κληρονομικότητα & πολυμορφισμός  
  - χειρισμός εξαιρέσεων & File I/O  
  - αντικειμενοστραφής C++ και χρήση STL  
- λειτουργήσει ως σημείο αναφοράς για τη δική μου πρόοδο  
- βοηθήσει όσους θέλουν να δουν πώς προσεγγίζονται εργασίες αυτού του μαθήματος

Οι ασκήσεις αντικατοπτρίζουν το επίπεδο κατανόησής μου κατά το **2ο εξάμηνο** των σπουδών μου.
