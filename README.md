### 2. Java Lab Set 2

**Focus:**  
- Classes, inheritance and polymorphism

Location: `labs/java-set-2/`

### 3. C++ Lab Set 3

**Focus:**  
- Basic C++ syntax and I/O  
- Arrays and multidimensional arrays  
- Classes, constructors, access specifiers (`private`, `protected`, `public`)  
- Operator overloading  
- Inheritance and virtual functions  
- STL `vector` and iterators   

Location: `labs/cpp-set-3/`
